# CloudPOS

Cloud-based point-of-sale system built with Java, Spring Boot, React and PostgreSQL.

## Project objective

CloudPOS is a modular point-of-sale system designed to manage:

- Products
- Stores
- Inventory by store
- Inventory movements
- Sales
- Cash payments
- Authentication and authorization
- Sales reports

## Technology stack

### Backend

- Java 21
- Spring Boot
- Spring Data JPA
- Spring Security
- PostgreSQL
- Flyway
- Maven

### Frontend

- React
- TypeScript
- Vite

### DevOps

- Docker
- Docker Compose
- GitHub Actions
